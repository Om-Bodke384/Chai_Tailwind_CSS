Chai Tailwind CSS is a lightweight utility-first styling system built from scratch in JavaScript. It reads class names that start with `chai-` and applies inline styles directly to your HTML elements.

Overview

What it is

Utility-first classes

Instead of writing CSS rules, you compose styles with classes like `chai-p-20`, `chai-bg-blue`, `chai-flex`, and `chai-cols-3`.

Runtime parser

On `DOMContentLoaded`, the script scans elements whose class attribute contains `chai-` and converts those utilities into inline styles.

Best use case

This works well for simple sites, experiments, demos, and learning how utility systems map classes to styles.

Setup

How to use it

1. Add the CDN script

<script src="https://cdn.jsdelivr.net/gh/Om-Bodke384/Chai_Tailwind_CSS@main/index.js"></script>

2. Use `chai-*` classes in HTML

<div class="chai-bg-red chai-textcol-white chai-p-20 chai-br-12">
  Card content
</div>

Current behavior is based on the present `index.js` parser: classes are read once on page load, so dynamically-added classes will need manual reprocessing if you want them applied later.

Reference

Available utilities

Category Class Result
Colors chai-bg-{color} Sets `background-color`
Colors chai-textcol-{color} Sets text color
Spacing chai-p-{number} Sets padding in pixels
Spacing chai-m-{number} Sets margin in pixels
Size chai-w-{number} Sets width in pixels
Size chai-h-{number} Sets height in pixels
Typography chai-fs-{number} Sets font size in pixels
Typography chai-text-{alignment} Sets text alignment
Border chai-br-{number} Sets border radius in pixels
Flex chai-flex Sets `display: flex`
Flex chai-center Centers content with flex
Flex chai-between Flex with `space-between`
Flex chai-col Sets flex direction to column
Grid chai-grid Sets `display: grid`
Grid chai-cols-{number} Creates repeated grid columns
Grid chai-rows-{number} Creates repeated grid rows
Grid chai-gap-{number} Sets gap in pixels
