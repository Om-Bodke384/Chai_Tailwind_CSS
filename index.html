What is Chai Tailwind CSS?

Chai Tailwind CSS is a lightweight styling library. It reads class names that start with chai- and applies styles directly to your HTML elements.

It is useful for simple websites, practice projects, and learning how utility classes work.

How to Use

Add this CDN link inside your HTML file:

<script src="https://cdn.jsdelivr.net/gh/Om-Bodke384/Chai_Tailwind_CSS@main/index.js"></script>

Then write classes like this:

<div class="chai-bg-red chai-textcol-white chai-p-20 chai-br-12">
  Hello World
</div>
