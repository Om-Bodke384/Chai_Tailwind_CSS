The above UI is rendered using a lightweight utility-first styling system built from scratch in JavaScript. It parses custom class names and applies styles dynamically without traditional CSS

`{number}` = any value (e.g., 5, 10, 100)  
`{color}` = red, blue, etc.
## ✨ Utilities

### 🎨 Colors
- chai-bg-{color}
- chai-textcol-{color}

### 📦 Spacing
- chai-p-{number}
- chai-m-{number}

### 📏 Size
- chai-w-{number}
- chai-h-{number}

### 🔤 Typography
- chai-fs-{number}
- chai-text-{alignment}

### 🧱 Border
- chai-br-{number}

### ⚡ Flex
- chai-flex
- chai-center
- chai-between
- chai-col

### 🧩 Grid
- chai-grid
- chai-cols-{number}
- chai-rows-{number}
- chai-gap-{number}
